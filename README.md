# RestrictorTray

Automatic joystick restrictor switching for arcade systems.

RestrictorTray monitors launched games and automatically switches your joystick restrictor between 4-way and 8-way modes when required.

Designed for use with modern arcade frontends and multi-system setups.

---

## Supported Hardware

GRS TOS Switchable 4-to-8-Way Restrictor

---

## Supported Systems

- MAME
- RetroArch
- PC Games
- TeknoParrot

---

## Features

- Automatic 4-way / 8-way restrictor switching
- Multi-system support
- Built-in graphical configuration interface
- Per-game override support
- MAME XML-based 4-way detection
- RetroArch content detection
- PC executable support
- TeknoParrot profile detection
- COM port configuration tools
- Integrated restrictor test controls
- Tray-based background operation

## Quick Start

1. Run `RestrictorTray.exe`
2. Select the TOS428 installation folder when prompted
3. Configure your MAME executable
4. Click **Rebuild XML**
5. Launch a game
